# References & Papers

## Papers

### RAG & Retrieval
- **"Retrieval-Augmented Generation for Knowledge-Intensive NLP Tasks"** (Lewis et al., 2020) - Original RAG paper
- **"Self-RAG: Learning to Retrieve, Generate, and Critique through Self-Reflection"** (Asai et al., 2023) - Adaptive retrieval
- **"Max-Min Semantic Chunking"** (2025) - AMI 0.90 clustering efficiency for chunking
- **"PIC: Pseudo-Instruction Chunking"** - Uses doc summaries to guide chunk grouping

### Embeddings
- **"BGE M3-Embedding"** (BAAI) - Multi-Lingual, Multi-Functionality, Multi-Granularity
- **"Qwen3 Technical Report"** - Qwen3-Embedding architecture
- **MTEB Benchmark** (Muennighoff et al.) - Embedding evaluation framework

### Local LLMs
- **"Gemma 3 Technical Report"** (Google, 2025) - Architecture, benchmarks, QAT
- **"Llama 3 Herd of Models"** (Meta, 2024) - Llama 3 architecture
- **"Qwen2.5 Technical Report"** (Alibaba, 2024) - Qwen2.5 architecture

### Document Understanding
- **"Docling"** (IBM Research) - ML-based document parsing
- **"Marker"** (VikParuchuri) - Layout-faithful PDF extraction

### TTS
- **"Kokoro TTS"** - Fast, lightweight text-to-speech
- **"Bark"** (Suno) - Transformer-based TTS with non-speech sounds
- **"Piper"** - Fast, local TTS
- **"Coqui XTTS v2"** - Cross-lingual voice cloning TTS
